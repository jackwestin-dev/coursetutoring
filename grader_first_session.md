Strategy Focus (From Today's Session)

1. Physics Problem Approach
   - Use unit analysis: If you need meters, and you have seconds, multiply by meters/second
   - Formula: Distance = Velocity x Time (d = v x t)
   - Scientific notation: When multiplying, add exponents (10^-3 x 10^2 = 10^-1)
   - If math is too complex: Guess, flag, move on—math questions are time sinks

2. Passage Triage (Chem/Phys)
   - Experimental passages: Look for "to determine..." or "an experiment was conducted"—read for IV/DV
   - Reference table passages: No experiment described—skip reading, go straight to questions

3. Equilibrium and Ksp
   - Q > Ksp: Precipitate forms (kidney stones form)
   - Q < Ksp: Solute dissolves
   - Le Chatelier's: System shifts to oppose change

4. Acid-Base (Henderson-Hasselbalch)
   - pH = pKa + log([A-]/[HA])
   - pH < pKa: More protonated (HA)
   - pH > pKa: More deprotonated (A-)
   - Aspartic acid/glutamate pKa is approximately 4 (acidic sidechains)

5. Approach to Amino Acids
   - Know structures, charges at physiological pH
   - pKa values will be given if needed—don't memorize exact numbers
