# SOP: Tutor Tasks & Session 1 Requirements
## Purpose
This SOP defines required tutor actions **before** and **during** the first tutoring session.
Cursor Agent should use this document to verify whether a tutoring transcript and related artifacts
demonstrate SOP compliance.

---

## Scope
Applies to:
- All 1:1 MCAT tutoring students
- All tutors completing Session 1
- Prep hour eligibility and Session 1 execution

---

## Definitions
- **Transcript**: Recorded tutoring session (video/audio → text)
- **Artifacts**: Google Docs, schedules, notes referenced or shared during session
- **Basecamp**: Source of student onboarding and diagnostic data

---

## TUTOR TASKS BEFORE SESSION 1 (Prep Hour Required)

### 1. Student Review (Required)
Tutor must review **all** of the following from Basecamp:

- Onboarding Survey Responses
- Content Diagnostic Results
- Baseline MCAT Score (with section breakdown)
- Passage Breakdowns
  - Science
  - CARS

**Agent Check**
- Transcript references student baseline score OR
- Tutor explicitly states familiarity with diagnostics/onboarding
- Tutor references Basecamp data

---

### 2. Documentation Preparation (Required)

Tutor must prepare student-facing documentation including:

#### A. Central Documentation
- Study Schedule (central reference doc)
- Passage Strategy notes

#### B. Strategy Documentation
Must include:
- Charts
- Dense Pathways
- Buzzwords
- Question Strategies:
  - 50/50 elimination
  - Rephrasing the question
  - Identifying content vs passage-based questions

#### C. Planning Assets
- Weekly topic assignment list (based on general student availability)
- Named passage strategy items student should work on
- Google Doc created for student reference

#### D. Documentation Must Include
- Student baseline score
- Agenda for Session 1
- Notes student can take home
- Study schedule

**Agent Check**
- Tutor mentions creating or preparing a document
- Tutor references strategies by name
- Tutor references baseline score or agenda
- Tutor references a schedule or plan

---

## DURING FIRST TUTORING SESSION

### 1. Shared Documentation (Required)

Tutor must create and share a Google Doc that includes:
- Weekly study schedule framework
- Alignment with live course curriculum
- Shared with:
  - anastasia@jackwestin.com
  - michaelmel@jackwestin.com

**Agent Check**
- Tutor explicitly says document will be shared
- Tutor references a Google Doc or shared notes

---

### 2. Study Schedule Requirements

#### A. Course-Based Schedule
- 515+ (Accelerated) pre-filled course schedule (if applicable)

#### B. Weekly & Daily Schedule
- Weekly content **on top of class materials**
- Daily schedule ONLY until next tutoring session (2-week horizon max)
- Focus on student-preferred resources (from survey)
- Must include:
  - Chapters to review
  - Number of application questions

**Agent Check**
- Tutor limits detailed planning to ~2 weeks
- Tutor asks about or references student resource preferences
- Tutor assigns specific content or question counts

---

### 3. AAMC Passage Work (Required)

Tutor must:
- Select an AAMC passage from:
  - Free Sample Test (preferred), OR
  - Q Packs
- Assign a passage task **at the beginning of the session**
- Use it to assess whether prior passage guidance was learned

**Agent Check**
- Tutor assigns a passage early in the session
- Tutor references AAMC material
- Tutor checks student understanding mid-session
- **Dual-source rule:** If the student notes document confirms AAMC materials were assigned/scheduled (e.g., "yes" to all AAMC documents), credit the tutor even if the transcript does not mention every AAMC assignment explicitly

### 3b. AAMC Question Packs/Resources Scheduling (Conditional)

If the student has AAMC question packs/resources (noted on student notes sheet), the tutor must schedule them. The specific AAMC resources:
1. MCAT Biology Question Pack, Volume 1
2. MCAT Biology Question Pack, Volume 2
3. MCAT Chemistry Question Pack
4. MCAT Physics Question Pack
5. MCAT Critical Analysis and Reasoning Skills Question Pack, Volume 1
6. MCAT Critical Analysis and Reasoning Skills Question Pack, Volume 2
7. MCAT Section Bank (Biology, Chemistry, Psychology/Sociology sections)
8. MCAT Official Prep Hub Independent Question Bank (formerly "Official Guide" questions)
9. MCAT Official Prep Critical Analysis and Reasoning Skills Diagnostic Tool
10. MCAT Official Prep Flashcards

**Note:** These are SEPARATE from full-length exams. "AAMC exams" are full-length exams (see Section 6). "AAMC question packs/resources" are the items listed above.

**Agent Check**
- If student has AAMC resources: Tutor schedules them with deadlines
- If student does NOT have AAMC resources: No penalty — full credit given automatically
- Evidence can come from either transcript or student notes

---

### 4. Review Methodology (Required)

Tutor must focus on:
- HOW to review passages
- Application of Bloom’s Taxonomy
- Deep understanding vs surface review

**Agent Check**
- Tutor references Bloom’s Taxonomy
- Tutor discusses review process (not just answers)

---

### 5. Scheduling & Tools

#### A. Topic Assignment
- Generate weekly topic assignment list

#### B. Daily Schedule
- Generated using student-preferred resources

#### C. Anki (If Applicable)
- Tutor must explain HOW to use Anki properly

**Agent Check**
- Tutor references Anki usage or learning tools
- Tutor explains method, not just recommendation

---

### 6. Full-Length Practice Exam Scheduling (Required — Mandatory for All Students)

- Total exams required: **11 (including baseline)**
- Tutor must schedule **10 future full-length practice exams** on the student notes sheet
- Sources: **Jack Westin Full Length 1–6** and **AAMC exams** (remaining to total 10)
- Each exam should have: Exam #, Source (JW FL or AAMC), Planned Date

**Note:** These full-length exams are SEPARATE from AAMC question packs/resources. "AAMC exam" = full-length exam. "AAMC question pack" = separate resource (see Section 3b).

**Exceptions**
- June Intensive / May 515 → Minimum 8 exams
- September 515 → 10 exams total (including baseline)

**Agent Check**
- Tutor references total exam count
- Tutor acknowledges exception if applicable
- Evidence can come from either transcript or student notes

---

### 7. Next Session Planning (Required)

- Tutor schedules or discusses next tentative session
- Based on student availability

**Agent Check**
- Tutor references next session timing or plan

---

## Compliance Outcome

### Pass
- Majority of required items explicitly mentioned
- Evidence of preparation + structured planning

### Flag for Review
- Missing baseline discussion
- No documentation mentioned
- No schedule created
- No AAMC passage work
- No full-length exam planning
- Student has AAMC question packs but no scheduling evidence

---

## Notes for Cursor Agent
- Exact wording is not required
- Conceptual alignment is sufficient
- Absence of documentation references = non-compliance
