# SOP: 515+/Intensive Tutor Tasks & Session Requirements

## Purpose
This SOP defines required tutor actions **before**, **during**, and **after** 515+/Intensive tutoring sessions (Sessions 1-3). The grading agent uses this document to verify whether a tutoring transcript and related artifacts demonstrate SOP compliance.

---

## Scope
Applies to:
- All 1:1 MCAT tutoring students in 515+ Course or Intensive track
- All tutors completing Sessions 1, 2, or 3
- Prep hour eligibility and session execution

---

## Definitions
- **Transcript**: Recorded tutoring session (video/audio → text)
- **Artifacts**: Google Docs, schedules, notes referenced or shared during session
- **Basecamp**: Source of student onboarding and diagnostic data
- **Student Notes Template**: Standardized Google Sheet for session documentation

---

## PRE-SESSION TASKS (All Sessions)

### 1. Student Review (Required)
Tutor must review from Basecamp:
- Onboarding Survey Responses
- Content Diagnostic Results
- Baseline MCAT Score (with section breakdown)
- Passage Breakdowns (Science and CARS)
- Prior FL scores (Sessions 2-3)

**Agent Check:**
- Pre-Session Notes section filled out before session starts
- Evidence of reviewing survey data, diagnostic scores, FL scores

### 2. Documentation Preparation (Required)
Tutor must prepare student-facing documentation:
- Study schedule in Google Sheet
- Strategy notes
- Weekly topic assignment list
- Student notes template properly named and linked

**Agent Check:**
- Student notes template named: "Student First/Last Name - Course Tutoring Note (Tutor Name)"
- Link set to JackWestin.com group view
- Student Overview tab populated

---

## DURING SESSION TASKS

### 3. Session Structure & Timing (Required — All Sessions)
Session must follow SOP time blocks:
- 0-5 min: Intro/check-in
- 5-15 min: Review (prior session, FL review for S2/S3)
- 15-70 min: Strategy portion (CARS and/or science)
- 70-90 min: Wrap-up (progression plan, next steps, takeaways, questions)

**Agent Check:**
- Session follows prescribed time allocation
- All required agenda items covered
- Wrap-up section not skipped or rushed

### 4. Study Schedule & Exam Planning (Required — Session 1 Primary)

#### A. Study Schedule
- Personalized study schedule created in Google Sheet with real dates
- Link shared with student
- Verified via SOP input or student notes

#### B. Full-Length Practice Exam Scheduling (Required — All Students)
- Total: **10 future full-length practice exams** scheduled
- Sources: **Jack Westin Full Length 1–6** and **AAMC exams**
- Each exam should have: Exam #, Source, Planned Date

**CRITICAL — Default/Placeholder Date Detection:**
The student notes sheet has a default/placeholder date of **"March 5"** (or "Mar 5", "3/5", "03/05") pre-filled in the Planned Date column. If exams still show this date, the tutor did NOT actually schedule them. **Exams with March 5th as the Planned Date do NOT count as scheduled.** Only exams with dates OTHER than March 5th count. If all/most exams show March 5th → treat as not scheduled (zero credit).

March 5th variations to detect: "March 5", "Mar 5", "3/5", "03/05", "March 5th", "Mar 5th"

**Agent Check:**
- Tutor references total exam count
- Evidence can come from either transcript or student notes
- Verify that exam dates are NOT the March 5th default/placeholder date

#### C. AAMC Question Packs/Resources Scheduling (Conditional)
If the student has AAMC question packs/resources, the tutor must schedule them. The 10 AAMC resources:
1. MCAT Biology Question Pack, Volume 1 & 2
2. MCAT Chemistry Question Pack
3. MCAT Physics Question Pack
4. MCAT CARS Question Pack, Volume 1 & 2
5. MCAT Section Bank (Bio, Chem, Psych/Soc)
6. MCAT Official Prep Hub Independent Question Bank
7. MCAT Official Prep CARS Diagnostic Tool
8. MCAT Official Prep Flashcards

**Note:** These are SEPARATE from full-length exams. "AAMC exams" = full-length exams. "AAMC question packs/resources" = the items listed above.

**Agent Check:**
- If student has AAMC resources: Tutor schedules them
- If student does NOT have AAMC resources: No penalty — full credit given automatically
- Evidence can come from either transcript or student notes

### 5. Passage Practice (Required — All Sessions)
- 3+ questions reviewed together with student
- Student teaches passage back (not tutor)
- Constructive feedback provided ~once per paragraph
- Student creates their own passage map; tutor provides feedback, not the map

**Agent Check:**
- At least 3 questions gone through
- Student reads aloud, explains reasoning
- Tutor provides paragraph-level feedback

### 6. Coaching & Questioning (Required — All Sessions)
- Tutor uses probing questions vs. lecturing
- Student does most of the talking
- Tutor clearly identifies missed question reasons
- Tutor doesn't let excuses ("I just guessed", "silly mistake") slide
- Tutor provides actionable, specific strategies
- Both timing AND accuracy addressed

**Agent Check:**
- Probing questions present: "What do you think?", "Why did you pick B?"
- No long monologues (>2 min without student engagement)
- Specific error types named

### 7. Session Wrap-Up (Required — All Sessions)
- Tutor asks student for main takeaway(s) near session end
- Tutor asks if student has questions/concerns
- Next steps clearly communicated

**Agent Check:**
- Takeaway question in last 20% of transcript
- Explicit check for questions before ending

## SESSION-SPECIFIC REQUIREMENTS

### Session 1 Only
- Booking/Calendly link shared for next session — 3 pts
- 2+ JW strategy course videos recommended for before next session — 3 pts

### Session 2 Only
- FL review process coached: student explains how they reviewed their FL — 4 pts
- Blind review principles discussed

### Session 3 Only
- Test-day toolkit verbalized by student: student demonstrates troubleshooting toolkit for time pressure — 4 pts

---

## POST-SESSION TASKS (All Sessions)

### 8. Documentation Completion (Required)
- In-session notes completed (within 24 hours)
- Notes shared with directors (Molly, Anastasia, Carl)
- Exam Progress tab updated
- Next steps clearly written in notes (not just verbal)
- Student note link added to Activity Completion Tracking (Column M)

**Agent Check:**
- In Session Notes section completed
- Evidence of sharing with directors
- Written next steps in notes

## SOP Verification Inputs (Third Source of Truth)

The grader UI may provide manual SOP verification inputs:

| Field | What It Confirms | Scoring |
|-------|-----------------|---------|
| Study schedule in Google Sheet | Schedule created and shared | YES = full credit, PARTIAL = 50%, NO = 0 (unless transcript/notes confirm) |
| AAMC question packs assigned | AAMC packs scheduled | YES = full credit, PARTIAL = 50%, NO = 0 (unless transcript/notes confirm) |
| Ten full-length exams assigned | All 10 FLs scheduled | YES = full credit, PARTIAL = 50%, NO = 0 (unless transcript/notes confirm) |

**Rules:**
- These inputs are a fail-safe — human confirmation when AI might miss something.
- Any source (transcript, student notes, OR SOP verification) confirming an item can override a "No" from another source.
- If SOP verification says YES but transcript/notes say otherwise, trust the human input (YES wins).

---

## Professionalism Standards (All Sessions)

- Calm, confident, supportive demeanor (no dismissive language)
- Don't guarantee outcomes or promise scores
- Refer students to proper channels for pricing/hours: "Chat with an advisor"
- Session starts/ends on time (open Zoom 5 min early)
- Communication on approved platforms only (no personal phone/social media)

---

## Notes for Agent
- Exact wording is not required
- Conceptual alignment is sufficient
- Absence of documentation references = non-compliance
- Most items are BINARY: full points or zero
- Three sources of truth: transcript, student notes, SOP verification inputs
